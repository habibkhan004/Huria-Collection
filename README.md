# HURIA COLLECTION

HURIA COLLECTION is a full-stack MERN ecommerce platform focused on cosmetics and footwear.  
It provides a modern shopping experience with secure authentication, product management, cart functionality, and order processing.

---

## 🚀 Tech Stack

### Frontend
- React.js
- Redux Toolkit
- React Router DOM
- Axios
- Tailwind CSS / CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs

---

## ✨ Features

### User Features
- Register & Login
- JWT Authentication
- Browse cosmetics & shoes
- Product details page
- Add to cart
- Checkout system
- Order history
- Responsive design

### Admin Features
- Admin dashboard
- Add / Update / Delete products
- Manage users
- Manage orders
